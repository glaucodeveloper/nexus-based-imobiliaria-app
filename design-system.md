# SuaImobiliaria Design System

Based on `proposta.png`, translated into tokens for the current hash-routed JS app.

## Personality
- editorial
- premium
- warm
- grounded
- high-trust

## Palette
- `--tone-sand-50` `#fcfaf4`: page glow and light backdrop
- `--tone-sand-100` `#f4eee1`: soft surfaces
- `--tone-sand-200` `#e9dfca`: quiet borders and fills
- `--tone-clay-500` `#b47e36`: primary accent
- `--tone-clay-600` `#9f6a27`: pressed / darker accent
- `--tone-forest-800` `#13201f`: premium dark UI
- `--tone-forest-900` `#0b1416`: hero / dashboard depth
- `--tone-ink-900` `#161818`: primary text
- `--tone-ink-500` `#6f776f`: secondary text

## Typography
- Display: `Cormorant Garamond`
- Body/UI: `Plus Jakarta Sans`
- H1/H2 should feel editorial, not product-marketing generic
- Metadata should be tight, uppercase, and quiet

## Spacing & Shape
- Cards: `24px` radius
- Large panels: `30px` radius
- Premium spacing favors larger vertical rhythm and asymmetry

## Motion
- `--motion-fast`: controls, nav, hover
- `--motion-medium`: cards, image zoom, panel emphasis

## Component Direction
- Topbar: dark glass capsule with restrained underline active state
- Hero: cinematic and text-heavy, not generic SaaS
- Listing: dense, broker-like catalog with soft luxury
- Detail: gallery first, commercial panel second
- Dashboard: dark shell + bright workspace, strong section separation

## Avoid
- Purple accents
- Flat white cards everywhere
- Identical padding across every section
- Generic fade-up-only feel
- Default “AI landing page” gradients
